# List of projects

List of projects

## Usage

``` r
get_all_projects(token = get_todoist_api_token())
```

## Arguments

- token:

  todoist API token

## Value

list of all projects

## Examples

``` r
if (FALSE) { # \dontrun{
# Set API key first
set_todoist_api_token()
# Get all projects
projects <- get_all_projects()
} # }
```
