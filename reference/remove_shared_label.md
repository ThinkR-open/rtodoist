# Remove a shared label

Remove a shared label

## Usage

``` r
remove_shared_label(name, verbose = TRUE, token = get_todoist_api_token())
```

## Arguments

- name:

  name of the shared label to remove

- verbose:

  boolean that make the function verbose

- token:

  todoist API token

## Value

NULL (invisible)

## Examples

``` r
if (FALSE) { # \dontrun{
remove_shared_label("label_name")
} # }
```
