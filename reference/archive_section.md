# Archive a section

Archive a section

## Usage

``` r
archive_section(section_id, verbose = TRUE, token = get_todoist_api_token())
```

## Arguments

- section_id:

  id of the section

- verbose:

  boolean that make the function verbose

- token:

  todoist API token

## Value

id of the archived section (invisible)

## Examples

``` r
if (FALSE) { # \dontrun{
archive_section("12345")
} # }
```
