# Get a single task by ID

Get a single task by ID

## Usage

``` r
get_task(task_id, token = get_todoist_api_token())
```

## Arguments

- task_id:

  id of the task

- token:

  todoist API token

## Value

list with task details

## Examples

``` r
if (FALSE) { # \dontrun{
get_task("12345")
} # }
```
