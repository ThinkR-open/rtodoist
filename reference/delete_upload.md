# Delete an uploaded file

Delete an uploaded file

## Usage

``` r
delete_upload(file_url, verbose = TRUE, token = get_todoist_api_token())
```

## Arguments

- file_url:

  URL of the file to delete

- verbose:

  boolean that make the function verbose

- token:

  todoist API token

## Value

NULL (invisible)

## Examples

``` r
if (FALSE) { # \dontrun{
delete_upload("https://...")
} # }
```
