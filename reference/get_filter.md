# Get a single filter by ID

Get a single filter by ID

## Usage

``` r
get_filter(filter_id, token = get_todoist_api_token())
```

## Arguments

- filter_id:

  id of the filter

- token:

  todoist API token

## Value

list with filter details

## Examples

``` r
if (FALSE) { # \dontrun{
get_filter("12345")
} # }
```
