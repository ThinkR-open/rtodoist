# List of tasks

List of tasks

## Usage

``` r
get_tasks(token = get_todoist_api_token())
```

## Arguments

- token:

  todoist API token

## Value

list of all tasks

## Examples

``` r
if (FALSE) { # \dontrun{
# Set API key first
set_todoist_api_token()
# Get all tasks
tasks <- get_tasks()
} # }
```
