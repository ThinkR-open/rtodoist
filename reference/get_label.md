# Get a single label by ID

Get a single label by ID

## Usage

``` r
get_label(label_id, token = get_todoist_api_token())
```

## Arguments

- label_id:

  id of the label

- token:

  todoist API token

## Value

list with label details

## Examples

``` r
if (FALSE) { # \dontrun{
get_label("12345")
} # }
```
