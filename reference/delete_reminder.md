# Delete a reminder

Delete a reminder

## Usage

``` r
delete_reminder(reminder_id, verbose = TRUE, token = get_todoist_api_token())
```

## Arguments

- reminder_id:

  id of the reminder

- verbose:

  boolean that make the function verbose

- token:

  todoist API token

## Value

NULL (invisible)

## Examples

``` r
if (FALSE) { # \dontrun{
delete_reminder("12345")
} # }
```
