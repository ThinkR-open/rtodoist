# Unarchive a section

Unarchive a section

## Usage

``` r
unarchive_section(section_id, verbose = TRUE, token = get_todoist_api_token())
```

## Arguments

- section_id:

  id of the section

- verbose:

  boolean that make the function verbose

- token:

  todoist API token

## Value

id of the unarchived section (invisible)

## Examples

``` r
if (FALSE) { # \dontrun{
unarchive_section("12345")
} # }
```
