# Get a single section by ID

Get a single section by ID

## Usage

``` r
get_section(section_id, token = get_todoist_api_token())
```

## Arguments

- section_id:

  id of the section

- token:

  todoist API token

## Value

list with section details

## Examples

``` r
if (FALSE) { # \dontrun{
get_section("12345")
} # }
```
