# Set todoist API token

This function use keyring to store your token from your todoist profile.
To find your token from todoist website, use
[`open_todoist_website_profile`](https://thinkr-open.github.io/rtodoist/reference/open_todoist_website_profile.md)

## Usage

``` r
set_todoist_api_token(token)
```

## Arguments

- token:

  todoist API token

## Value

token
