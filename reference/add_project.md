# Add a new project

Add a new project

## Usage

``` r
add_project(project_name, verbose = TRUE, token = get_todoist_api_token())
```

## Arguments

- project_name:

  name of the new project

- verbose:

  boolean that make the function verbose

- token:

  todoist API token

## Value

id of the new project

## Examples

``` r
if (FALSE) { # \dontrun{
add_project("my_proj")
} # }
```
