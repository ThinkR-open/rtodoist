# Get a single project by ID

Get a single project by ID

## Usage

``` r
get_project(project_id, token = get_todoist_api_token())
```

## Arguments

- project_id:

  id of the project

- token:

  todoist API token

## Value

list with project details

## Examples

``` r
if (FALSE) { # \dontrun{
get_project("12345")
} # }
```
