# Leave a workspace

Leave a workspace

## Usage

``` r
leave_workspace(workspace_id, verbose = TRUE, token = get_todoist_api_token())
```

## Arguments

- workspace_id:

  id of the workspace

- verbose:

  boolean that make the function verbose

- token:

  todoist API token

## Value

NULL (invisible)

## Examples

``` r
if (FALSE) { # \dontrun{
leave_workspace("12345")
} # }
```
