# Get a single comment by ID

Get a single comment by ID

## Usage

``` r
get_comment(comment_id, token = get_todoist_api_token())
```

## Arguments

- comment_id:

  id of the comment

- token:

  todoist API token

## Value

list with comment details

## Examples

``` r
if (FALSE) { # \dontrun{
get_comment("12345")
} # }
```
